Setup Instruction
Backend Setup:
Navigate to the backend directory:
  cd Notes-App/Backend
Install dependencies:
  npm install
Configure environment variables: -> Create a .env file -> Update the environment variables with your MongoDB connection URI, JWT secret key
  npm start
Frontend Setup
Navigate to the frontend directory
  cd Notes-App/Frontend/notes-app
Install dependencies:
  nppm install
  
Start the frontend development server:
  npm run dev
